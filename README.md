OpenGenius

OpenGenius is an AI-powered web application built using Python, Django, and OpenAI API that helps users generate intelligent responses, assist in problem-solving, and enhance productivity through conversational AI. The platform provides a simple web interface where users can input queries and receive AI-generated answers in real time. It is designed with a scalable backend, secure API handling, and cloud deployment support, making it suitable for AI-based assistance, learning support, and productivity enhancement tools.
